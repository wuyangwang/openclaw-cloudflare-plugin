# openclaw-cloudflare-plugin

Cloudflare ts-sdk wrapper for OpenClaw.

## Features(Current)

- **KV**: KV curd.
- **D1**: D1 curd.
- **Worker**: show your Workers.

## Installation

```bash
# step 1
openclaw install openclaw-cloudflare-plugin
# step 2 input token
openclaw cloudflare login --token xxxx
# optional (check status)
openclaw cloudflare status
# done
```

## Usage

- 1 make sure you have config channel. like: Discord/Telegram/Wecom Bot（企业微信机器人）
- 2 open Discord and send message like this: "Please show my cloudflare kv list"
- 3 done
